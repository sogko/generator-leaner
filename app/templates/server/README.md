# ExpressJS Server Folder

Recommended folder to organize your ExpressJS server.

The ```server``` folder contains the following default components:

* [```app.js```](app.js)
* [```config.js```](config.js)
* [```middlewares/```](middlewares/)
* [```models/```](models/)
* [```routes/```](routes/)
* [```views/```](views/)
